Pernah dengar istilah "titip absen" atau karyawan yang pakai Fake GPS untuk clock in dari rumah?

Masalah ini nyata — dan saya memutuskan untuk membangun solusinya sendiri secara Open Source. 🚀

Perkenalkan PasPapan V2: platform Workforce Management & Payroll skala Enterprise yang saya rancang ulang dari nol menggunakan Laravel 11 & Livewire 3.

Bukan sekadar aplikasi ceklis masuk/pulang — ini sistem HR lengkap yang siap skala Enterprise.

Apa saja yang bisa dilakukan PasPapan?

🛡️ Keamanan Absensi Anti-Kecurangan
→ Smart Geofencing + deteksi Mock Location + Device Lock
→ Wajib Face ID enrollment sebelum bisa clock in — buddy punching mustahil
→ QR Code Attendance sebagai alternatif absensi cepat

📊 Performance Appraisal & KPI System
→ KPI hierarkis dengan bobot per kategori (Kinerja Utama 50%, Kepegawaian 50%)
→ Self-Assessment oleh karyawan + Evaluasi Subjektif oleh Manajer
→ Bell Curve otomatis untuk kalibrasi distribusi nilai (A/B/C/D/E)

💰 Payroll Otomatis (Regulasi Indonesia)
→ Perhitungan satu klik: Gaji Pokok, Tunjangan Harian, Lembur
→ Potongan otomatis: BPJS Kesehatan, JHT, JP, PPh 21 (TER)
→ Kasbon terintegrasi langsung ke siklus penggajian
→ E-Payslip terenkripsi dengan password pribadi karyawan

📦 Asset Lifecycle Management
→ Lacak perangkat kantor (laptop, kendaraan, gadget) dari pembelian hingga pengembalian
→ Status real-time: Assigned, Maintenance, Returned
→ Karyawan bisa ajukan Request Return langsung dari HP

🏢 Administrasi Korporat
→ Multi-branch dengan isolasi data per wilayah
→ Approval workflow bertingkat: Cuti, Lembur, Reimbursement
→ Jadwal shift, kalender libur, pengumuman internal
→ Export/Import data karyawan & absensi via Excel

Semuanya bilingual (EN/ID), PWA-ready, dan dark mode friendly.

Tech stack: Laravel 11 · Livewire 3 · Tailwind CSS · PWA · Face Recognition API

Saya membuka seluruh source code ini karena percaya: solusi HR yang baik seharusnya bisa diakses semua orang, bukan hanya perusahaan besar.
